Detailed Description

Library to control a VT100 terminal via an Arduino.

Using this library it is possible to make better use of VT100 capable terminal programs such as Tera Term. This library does not work when being used with the Serial Monitor of the Arduino IDE. The Serial Monitor does not support VT100 commands und will not react to them properly.
